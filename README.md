# Polygenic Risk Score in Familial Hypercholesterolemia

MSc Bioinformatics Thesis — UNIR (2025-2026)

Analysis of the predictive utility of a polygenic risk score 
in a monogenic FH cohort (N=1,200+ genotyped patients).

## Methods
R-based statistical workflows including GLMM with family frailty, 
Cox survival analysis, mediation analysis, and ROC curve comparison.

## Main packages
lme4, coxme, survival, survminer, pROC, gtsummary, ggplot2

## Note
Data not included (patient confidentiality).
